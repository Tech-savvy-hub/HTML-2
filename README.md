# Project Summary

This project is my first simple multi-page website created using HTML5.  
I built two connected webpages: a Home Page (`index.html`) and a Contact Page (`contact.html`).

Through this project, I practiced using HTML5 semantic elements, navigation links, forms, media elements, interactive elements, and iframes. The website includes information about me, my interests, my school, and my favorite hobby which is gaming.

I also added audio, video, a Google Map, a YouTube video, and a contact form with different input types such as radio buttons, checkboxes, dropdown menus, and textarea fields.

This project helped me improve my understanding of basic web development and how different HTML elements work together to create a functional website. Humanity invented websites and then made students explain them in README files. Beautiful cycle honestly.
